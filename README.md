# Dotfiles

My personal macOS/Linux config backup.

This repo stores my main setup so I can restore things quickly after a reinstall.

## What is backed up

- Kitty config
- Neovim / LazyVim config
- `.zshrc`
- Homebrew packages with `Brewfile`
